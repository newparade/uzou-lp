# UZOU ブランドガイド

## ブランドカラー（正式5色）

| 名称 | HEX | 用途例 |
|---|---|---|
| ライトティール | #8BC0CA | アクセント、ホバー、ハイライト |
| ミドルティール | #34626F | CTAボタン、リンク、バッジ |
| ディープティール | #2B4954 | 見出しアクセント、グラデーション終点 |
| ダークティール | #1F353E | ダークセクション背景、ナビ、フッター |
| ブラック | #040404 | テキスト最重要 |

### 背景色
- `#E8F2F4` — ティールベースの薄い背景
- `#F0F7F8` — さらに薄いティール（hover背景、カード内背景）

> **注意**: 上記は公式ブランドカラーであり、LP実装では必ず使用すること。
> ただし、追加の補助色やグラデーションは自由に設計してよい。

## フォント
- **日本語**: Noto Sans JP（Google Fonts）
- **英語**: 任意（Montserrat, Inter, Outfit等から選択可）

## ロゴ
- テキストロゴ: **UZOU**（太字）
- ロゴマーク: 「U」を角丸四角で囲んだもの（CSSで再現可）
- 会社名: 株式会社ヒトクセ
