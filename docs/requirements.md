# UZOU LP 要件定義書

## 1. サービス概要

### UZOU（ウゾウ）とは
株式会社ヒトクセが運営する広告プラットフォーム。広告主・代理店とWebメディアを直接つなぐ独自の商流基盤を活かし、レコメンド・インフィード・ディスプレイ広告を横断的に支援するサービス。

### ターゲット
1. **広告主・代理店**: 新規メディア開拓、CPA改善、成果のスケールを求める企業
2. **媒体主（Webメディア）**: 広告収益の最大化、新たな成長戦略を求めるメディア運営者

### 提供価値
- 広告成果とメディア収益の**双方を最大化**
- AIによる配信最適化
- 独自の商流基盤による直接取引
- 持続可能な広告流通の実現

## 3. セクション構成（上から順）

### ヘッダー

- CTA: お問い合わせ / 資料ダウンロード

### Hero（FV）
- メインコピー: **「メディア広告をもっとスマートに」**
- サブコピー: 「AIが導く、最適な広告×メディアマッチング。広告主の成果とメディアの収益、その両方を最大化する次世代プラットフォーム。」
- CTA: 資料ダウンロード / お問い合わせ
- 安心ポイント: 完全無料 / 30秒で完了 / 営業電話なし
- ビジュアル: ダッシュボードUI風のモック画像（月間収益、CTR等の数値表示）
- バッジ: IT Review LEADER / AI搭載

### About（UZOUとは）
- リードコピー: 「メディア・広告・ユーザーの**最適な循環**をつくる。」
- 説明文: デジタル広告市場の変化、UZOUの独自商流基盤、レコメンド・インフィード・ディスプレイ広告の横断支援
- フローチャート: 広告主・代理店 → UZOU Platform（AI最適化） → Webメディア

### Solution（課題解決セクション）— ダーク背景
- タイトル: 「その課題、UZOUが**解決**します」
- 2カラム構成:

**広告主の方（Advertisers）**
1. 新規メディア開拓の調整コストが高く、検証スピードが上がらない
2. 特定の媒体中心の運用から抜け出せず、成果改善の選択肢が広がらない
3. 成果のロジックや再現性を十分に説明できない
4. 独自性のある媒体提案ができず、競合との差別化に苦戦
5. CPAは維持できても、案件を安定的にスケールさせる打ち手が不足

**媒体主の方（Publishers）**
1. 収益が頭打ちとなり、次の成長戦略が描けていない
2. 規制やAIの影響で、広告単価や流入構造の将来が見通せない
3. 新施策を打ちたいが、検証リソースが不足している
4. 特定プラットフォーム依存から脱却できない
5. データを活かした収益改善が進んでいない

### CTA Strip
- 3つのCTAカード（横並び）:
  1. サービス紹介資料をダウンロードしたい
  2. 費用の見積もりやできることを知りたい
  3. UZOUのUI/UXを体験したい

### Features（特徴）
3つのFeatureブロック（左右交互配置）:

1. **AI最適化配信エンジン**
   - 独自AIが広告とメディアの最適マッチングをリアルタイムで実現
   - CTR・CVR・視認性を総合分析し、最適配信先を自動選定
   - 機械学習による配信精度の継続的改善

2. **高収益メディアネットワーク**
   - 厳選された優良メディアとの直接取引
   - 中間マージンを削減し、広告主・メディア双方の収益を最大化
   - 独自の商流基盤による安定的な広告供給

3. **ワンストップソリューション**
   - 広告の企画から配信、分析までをワンプラットフォームで完結
   - 専任担当者による運用サポート
   - リアルタイムダッシュボードで透明性の高いレポーティング

- 導入メディア数: **200+**
- 月間インプレッション: **3.5億+**
- 月間広告リクエスト: **50億+**

5. レポート・改善 — 定期レポートと改善提案

## 4. 技術要件

```

style.css   — 全スタイル（CSS変数、アニメーション、レスポンシブ）
script.js   — 全インタラクション（パーティクル、アニメーション、アコーディオン等）
```
