## ブリーフ分析

- **業種**: AdTech / ネイティブ広告配信プラットフォーム（BtoB SaaS）
- **運営会社**: 株式会社Speee（東証スタンダード 4499）
- **ターゲット**:
  - プライマリ: 広告代理店のメディアバイヤー（30-40代）。月間配信予算の裁量を持ち、メディアポートフォリオの多様化に迫られている。自分でも複数のLPを見て目が肥えている。陳腐なBtoB SaaS語彙を即座に見抜く。
  - セカンダリ: Webメディアの収益責任者。サードパーティCookie廃止後の収益多角化を模索中。タグ設置のリスク・手間に敏感。
- **目的**: 資料ダウンロード（主CTA） / お問い合わせ（副CTA）
- **KPI**: 資料DLコンバージョン率（目標2%台）、Hero通過率（70%以上）、離脱率（ファーストビュー以降で50%以下）
- **ブランドトーン**: 知的だが挑発的。「広告は嫌われている。UZOUはそれを変える」という強いスタンス。精密だが有機的。暗闇の中の光。
- **競合差別化**:
  - GMO系・Amazon DSP: 大手の信頼感だが無機質。UZOUは「メディアを選ぶ目利き」の存在感で差別化
  - B-Dash: コンセプト中心で情報が薄い。UZOUは500+という具体数値でリアリティを出す
  - 独自商流基盤（直接取引・中間マージンなし）は語りすぎず、ビジュアルとコピーの組み合わせで体感させる

## デザイン要件（v10固有）

- 「有象無象」ブランド哲学をデザイン全体で体現する（形あるものとないものの接続）
- セクションごとに異なるレイアウト構造・背景表現・アニメーションを持つ（均一禁止）
- ティールは「光の源」として戦略的に使う。7割はダーク/ライトのモノクロームが支配
- Heroの右半分に「UZOU Connection Visualizer」（Canvas）を配置。空虚にしない
- 数値セクション（Results）を非対称レイアウトで「規模感の衝撃」に仕立てる
- Solutionはタブ切替（Advertisers / Publishers）で情報過多を回避
- Testimonialsは中央1枚フィーチャー形式 + 数値（CPA改善率）を大きく
- CTA Stripは3列テーブル型を廃止。1主CTAに集中させる
- 禁止ワード: 「スマートに」「次世代」「AIが導く」「最適化」「双方を最大化」

### 03. Solution（課題共感）

**背景カラー**: `#1F353E`（ダークティール）+ ノイズテクスチャ + 右上アンビエントグロー

**遷移エフェクト**: Media Trust → Solution: `clip-path`による斜め境界線（角度5deg）

**コンテンツ構造**:
- キャプション: 「SOLUTION」
- H2コピー: 「その悩み、業界共通です。」
  - 根拠: 「悩み」はターゲットが自分ごと化しやすいワード。「業界共通」で孤独感を取り除く
- タブUI:
  - タブ1: 「広告代理店・広告主の方」（Advertisers）
  - タブ2: 「メディア運営者の方」（Publishers）
- Advertisersコンテンツ（5項目、番号付きリスト）:
  1. 新規メディア開拓に時間と調整コストがかかりすぎる
  2. 特定媒体への集中から抜け出せず、成果改善の打ち手が尽きている
  3. 配信ロジックの説明責任を果たせず、クライアントへの報告が苦しい
  4. 他社との差別化になる媒体提案ができていない
  5. CPAは守れてもスケールさせる方法論が見つからない
- Publishersコンテンツ（5項目）:
  1. 広告収益が頭打ち。次の成長策が見えない
  2. サードパーティCookie廃止後の収益構造が不安定になっている
  3. 新施策を試したいが、検証できるリソースがない
  4. 特定プラットフォームへの依存度を下げられない
  5. 持っているデータを収益につなげる方法がわからない
- 各項目にSVGアイコン（解決を暗示するシンプルな幾何学モチーフ）

**実装リソース**:

## Heroコピー提案（5案）

design-leadブリーフの3案を踏まえ、design-directorとして5案を提案する。

### 推奨: 案D「500のメディアが、接続されている。」

**推奨理由**: 数値による社会的証明（500）と、UZOUのコアバリュー「接続」を組み合わせた最もUZOU固有の表現。動詞「接続されている」が現在進行形の信頼感を生む。ターゲット（広告代理店）が一瞬で「500のメディアにリーチできる」と理解できる。7語以内。

---

**案A**: 「広告が、メディアを救う。」  
語数: 7語（日本語カウント）  
強み: 逆説的・インパクト最大。「広告は嫌われている」という常識への反論。design-leadが提案した案そのもの。  
弱み: やや強すぎて「救う」という主語が広告代理店には刺さりにくい可能性。媒体主向きの表現。  
サブコピー案: 「500以上のメディアと直接つながる配信基盤。UZOUは、広告とメディアの関係を変えてきた。」

**案B**: 「500のメディアが、接続されている。」 ← 推奨
語数: 10語（日本語カウント）/ 表示行: 2行  
強み: 数値社会証明ファースト。「接続」はUZOUのコアバリュー直結。現在形が持続的な状態を示す。  
弱み: 行動喚起が弱い（それはサブコピーで補う）  
サブコピー案: 「広告主とメディアを直接結ぶ独自の商流基盤。株式会社Speeeが運営する、国内最大級の広告配信プラットフォーム。」

**案C**: 「有象無象の広告市場に、流れをつくる。」  
語数: 11語  
強み: ブランド名の語源「有象無象」を直接使う。「流れをつくる」は哲学的かつ具体的。  
弱み: 「有象無象」という読みにくい漢字がスキャンを妨げる。初見ユーザーへの訴求が弱い。  
サブコピー案: 「広告主・代理店・メディアを直接つなぐ独自基盤。複雑な調整を、シンプルな成果に変える。」

**案D**: 「嫌われない広告を、500のメディアへ。」  
語数: 11語  
強み: 「嫌われない広告」という挑発的な言葉でUZOUのスタンスを示しつつ、「500のメディア」という数値で着地。  
弱み: 「嫌われない広告」の説明がないと意味が伝わりにくい。  
サブコピー案: 「コンテキストを読む配信技術で、ユーザーにとって自然な広告体験を実現。UZOUは、広告とメディアの間にある課題を変えてきた。」

**案E**: 「あなたのメディアに、次の1億が流れる。」  
語数: 13語  
強み: Publisher向けに特化。「1億」という金額の臨場感。「流れる」はブランドコンセプトと直結。  
弱み: Advertiser向けの訴求が薄くなる。統合1ページのHeroとしては片側に偏りすぎ。  
サブコピー案: 「500以上のメディアが選んだ理由がある。UZOUの直接取引基盤で、広告収益の構造を変える。」
