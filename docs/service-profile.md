# サービスプロファイル（Phase 1）

## 基本情報
- **サービス名**: UZOU（ウゾウ / 有象無象）
- **運営**: 株式会社ヒトクセ
- **サービスの種類**: toB（BtoB SaaS / AdTech）
- **ジャンル**: 広告プラットフォーム / DSP
- **プラットフォーム**: Web

## ターゲット
- **プライマリ**: 広告代理店のメディアバイヤー（30-40代）、事業会社のマーケティング担当者
- **セカンダリ**: Webメディアの収益責任者
- **利用シーン**: 新規メディア開拓、CPA改善、メディア収益最大化
- **ITリテラシー**: 高

## ブランド
### ロゴコンセプト
- 語源は「有象無象」＝「形あるもの・ないもの」
- Zのロゴに斜め線 → 「嫌われる広告に一石を投じる」意志
- 下部が溶けるデザイン → 「既存概念を溶かしていく」メタファー
- 数字の「1」を模した形状 → ナンバーワンの地位

### トーン&マナー
- プロフェッショナルだが堅すぎない
- データ・テクノロジー重視
- 信頼感と先進性のバランス
- **装飾**: 幾何学的・線形要素（バブル/丸ではない）
- **背景**: 白ベース基調

## 今回のデザイン対象
- **対象**: UZOU サービスLP v9
- **目的**: 資料ダウンロード / お問い合わせのコンバージョン獲得
- **KPI**: CVR（資料DL率）、直帰率低減
- **ページ構成**: 現時点では広告主＋媒体主の統合1ページ（将来的にページ分割の可能性あり）

## デザイン制約
- 管理画面UIの詳細表示は不要（コンセプト訴求優先）
- 過度なイラスト使用は避ける
- テンプレート感・AI生成感を排除
- 将来的に他サービス（Roots、AdRoom）へ展開可能な汎用性
- 「有象無象」コンセプトを装飾に反映
